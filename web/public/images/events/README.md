# Event images

Add event flyer/photo images here. The Events page will display them automatically.

**Currently included:**
- `ravi-bakhai.jpg` – Ravi Bakhai Guest Speaker Event
- `jarry-xiao.jpg` – Jarry Xiao Tech Talk

To add more, add the filename to `imagePath` in `src/components/events.json`. PNG works too—just update the extension.
